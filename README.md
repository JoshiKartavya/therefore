# Therefore Fashion - Website Redesign (Demo)

> A modern website redesign concept for Therefore Fashion built using React.js and Tailwind CSS.

---

## Overview

This project is a UI/UX redesign and frontend development concept for **Therefore Fashion**, a contemporary fashion brand. The objective is to create a fast, responsive, and visually engaging website that reflects the brand's premium identity while providing an exceptional user experience.

This is an independent demo project created for portfolio and presentation purposes.

## Installation

Clone the repository

```bash
git clone https://github.com/kartavya-joshi/therefore-fashion-demo.git
```

Go into the project

```bash
cd therefore-fashion-demo
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
